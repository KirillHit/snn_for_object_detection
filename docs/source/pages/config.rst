Файл конфигурации
=================

Позволяет настроить параметры сети, обучения и сценарий работы. Файл должен находится по пути ``config\config.yaml`` в директории пакета. 

.. include:: ../../../config/config.yaml
    :literal:

В файле конфигурации должны быть указаны следующие параметры:

==============  ==============
   Параметр        Описание   
--------------  --------------
Mode            ...
--------------  --------------
Dataset         ...
==============  ==============
